Gollum+GitHub搭建个人Wiki

2021-02-25

配置

1.新建一个GitHub Repository 命名为 xxx.github.io (不确定能否用其它命名方式，个人觉得应该可以)

2.打开已创建Repository的Setting页面勾选Features 栏目中的这两个选项 Wikis和Restrict editing to collaborators only(这两个选项默认是开启的该步骤可以跳过)

3.进入GitHub该Repository中的Wiki栏目，复制该栏目下Clone this wiki locally中的内容，把Wiki clone 到本地并进入该目录

4.终端执行gem install gollum一下是我按照gollum过程中遇到的错误

5.在wiki目录执行gollum命令

6.本地成功启动维基打开网址：http://0.0.0.0:4567/
