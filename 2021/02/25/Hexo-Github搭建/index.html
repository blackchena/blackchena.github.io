主题设置

我用的主题是这个大道至简，里面包含的主题替换的方法，注意:

  • 修改的根目录中的_config.yml文件中的theme值的为maupassant
  • 配置完成后执行hexo clean && hexo g -d（g 是 generate 缩写，d 是 deploy 缩写，部署前最好能先执行一下 hexo clean 命令，清除缓存文件 (db.json) 和已生成的静态文件 (public)。在某些情况（尤其是更换主题后），如果发现您对站点的更改无论如何也不生效，您可能需要运行该命令）
  • 重新部署后可能需要等一段时间博客才会更新新发布的内容，即使clean之后也一样
